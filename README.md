# XML Transformer
A library for transforming (A.K.A serializing/deserializing) XML structures.

## Installation
1. Add the dependency to your `shard.yml`:

```yaml
dependencies:
  xmlt:
    github: devnote-dev/xmlt
```

2. Run `shards install`

## Usage
```crystal
require "xmlt"
```
